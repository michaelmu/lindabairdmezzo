Read Me First

This page is intentionally simple and static. It does not call AWS from the browser. The data is exported into JSON during maintenance refreshes, then rendered at build time.

Because the visitor metric is based on distinct IP addresses, it will overcount in some cases and undercount in others. It is best used for directionality, not billing-grade analytics.

Device, browser, operating-system, and referrer cuts are inferred from CloudFront request metadata. Location is not included yet; adding country cleanly would require richer CloudFront log fields.

If you need deeper attribution or session analysis later, add a real analytics product. This page is for lightweight traffic visibility without changing the current site architecture.
